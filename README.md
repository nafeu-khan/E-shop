# Eshop
Basic E-commerce website.

## Feature 
1. Registration
2. Login
3. Product view
4. Add to cart
5. Purchase

## Technology 
### Frontend
    -Reactjs
    -Bootstrap
### Backend
    -Nodejs
    -Expressjs
### Database
    -Postgresql

## Run the application
for frontend:
```
    cd client
    npm install
    npm  run dev
```
for backend: 
```
    cd server 
    npm start
```
